Base de Dados
É utilizada para a criação e compartilhamento de um banco de informações na disciplina
Do ponto de vista pedagógico, como a ferramenta pode ser utilizada?

A base de dados pode ser utilizada para a socialização de conteúdos produzidos pelos alunos.

Os conteúdos armazenados podem ser imagens, sons, vídeos, textos e links, por exemplo.

Permite a validação dos materiais e a inserção de notas pelo professor
Comentários e avaliação

Permite também, quando configurada para isso, que os participantes coloquem comentários sobre cada registro e avaliem os trabalhos dos colegas.

Vídeo Tutorial de Configuração e Aplicabilidade

Dicas de Uso

É possível exportar uma planilha com as respostas da Atividade.

Ótima opção para fazer um mural de imagens.

Com um pouco de conhecimento em HTML é possível estilizar cada tipo de visualização de respostas.
