Chat
A ferramenta disponibiliza uma discussão síncrona em tempo real entre os estudantes e o professor
Do ponto de vista pedagógico, como a ferramenta pode ser utilizada?

Esta ferramenta contém diferentes temas para uso e uma interface acessível, para auxíliar o professor a manter a conversa com os estudantes.

Para acessar um chat, basta que o aluno clique no link de bate papo e digite suas mensagens. Além disso, é possível mencionar e chamar a atenção de outro participante.

Se configurado corretamente, os participantes de um chat podem ver o histórico de conversas passadas, desde que o chat não tenha sido deletado.
Recursos de um Chat

Ao usar um chat, os participantes tem à disposição recursos como emotes, possibilidade de adicionar links e beeps, onde este último funciona como uma chamada de atenção.
